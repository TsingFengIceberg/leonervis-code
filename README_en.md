# Leonervis Code

[中文](README.md)

Leonervis Code is a lightweight, local, single-user coding-agent CLI prototype. It aims to implement a small, mini-scale set of core coding-assistance capabilities inspired by tools such as Claude Code.

The project does not aim to reproduce, replace, or claim feature parity with any existing product. It begins with practical Harness foundations: controlled model calls, tool execution, workspace boundaries, and permission approval, building toward a coding agent that is understandable and verifiable.

Before starting this project, the related Harness study and reading notes were collected in [Harness-study](https://github.com/TsingFengIceberg/Harness-study). Leonervis Code is a practical implementation exercise built on that learning and documentation.

Repositories under `learning-submodules/` are read-only learning references. They are not runtime dependencies and must not be imported by product code.
