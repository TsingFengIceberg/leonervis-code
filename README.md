# Leonervis Code

[English](README_en.md)

Leonervis Code 是一个面向本地单用户使用的轻量级 Coding Agent CLI 原型，计划实现 mini 级、类似 Claude Code 的核心编码辅助能力。

项目当前不追求复刻或替代任何现有产品，而是从受控的模型调用、工具执行、工作区边界与权限确认等 Harness 基础能力开始，逐步实践一个可理解、可验证的 Coding Agent。

在启动这个项目之前，相关的 Harness 学习与阅读笔记记录在 [Harness-study](https://github.com/TsingFengIceberg/Harness-study) 仓库中；Leonervis Code 是在这些学习与记录基础上的实践尝试。

`learning-submodules/` 中的仓库仅供只读学习，不是本项目的运行时依赖，也不能被产品代码导入。
